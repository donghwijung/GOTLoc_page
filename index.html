GOTLoc: General Outdoor Text-based Coarse Localization Using Scene Graph Retrieval with OpenStreetMap

Donghwi Jung, Keonwoo Kim, Seong-Woo Kim

Seoul National University
Preprint Code Video

Abstract

We propose GOTLoc, a robust coarse localization method designed for outdoor environments where GPS signals are unavailable. Unlike existing approaches that use point cloud maps, which are large and difficult to store, GOTLoc leverages scene graphs generated from text descriptions and maps for coarse localization. This method improves scalability by replacing point clouds with compact data structures, allowing robots to efficiently store and utilize extensive map data. Additionally, GOTLoc eliminates the need for custom map creation by using publicly available OpenStreetMap data, which provides global spatial information. We evaluated its performance using the KITTI360Pose dataset with corresponding OpenStreetMap data, comparing it to existing point cloud-based coarse localization methods. The results show that GOTLoc achieves comparable ac- curacy while significantly reducing storage requirements. In city- scale tests, it completed processing within a few seconds, making it highly practical for real-world robotics applications.

Concept

Concept of GOTLoc. The user describes the surrounding environment in text and provides it to the delivery robot. The robot interprets the text description to determine the user's scene. In this process, both the text description and the OSM data are converted into scene graphs. The robot then retrieves and identifies the most similar pair, estimating the current scene based on the highest similarity.

Process

Process of GOTLoc. The method consists of three sequential steps: 1) Scene graph generation, 2) Scene graph candidates extraction, 3) Scene graph retrieval and scene selection. The input data are a query text description and OSM. The output is the matching scene id.
